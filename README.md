(Note: dinput8.dll is not open source here so download it at your own risk!)

What is added

- Supports both 2048x1024 and 4096x2048 image sizes (4096x2048 better quality)
- Works both on world planner and renderworld images (renderworld worse quality)
- Opacity and toggle setting found in in-game settings

What is missing:

- Hotkey implementation
- Fix for 125x75 and other size dimension worlds
- Removal of world background for less view obstruction (workaround is using weather machine - stuff, or nothingness weather)

Tutorial:

Download (don't screenshot!) an image from either the world planner or a growtopia renderworld.

<img width="1525" height="886" alt="Screenshot 2026-03-02 162806 (1)" src="https://github.com/user-attachments/assets/705d2633-4ad3-4ffa-b45e-1a7a2a3bab4f" />

Import downloaded image to photopea and export file, changing the image dimensions to either 2048x1024 or 4096x2048 as PNG (unlink link image ratio/dimension sizes, the button between width and height)

<img width="1338" height="892" alt="Screenshot 2026-03-02 163258" src="https://github.com/user-attachments/assets/ffb04b3a-6cd0-47e4-a602-4a35cbe676ae" />

Go to https://tools.cernodile.com/rttex.html, pick the image to RTTEX option, and convert the downloaded photopea image into .RTTEX file by pressing the yellow/green convert button

<img width="1760" height="975" alt="Screenshot 2026-03-02 164002" src="https://github.com/user-attachments/assets/b8b2ea2a-b1ad-42b8-a7ab-80fe456acb6b" />

Once you have the rttex file, rename it to "overlay"

<img width="770" height="113" alt="Screenshot 2026-03-02 164702" src="https://github.com/user-attachments/assets/5cde38b6-0587-4a76-9ddd-2acd561b8fab" />

Open your growtopia folder (with "open file location on growtopia icon or WIN+R and type %localappdata% and find) and go to game folder and drop the file there (not on top of another file)

<img width="886" height="717" alt="Screenshot 2026-03-02 165111" src="https://github.com/user-attachments/assets/51ffa0b0-a199-43e0-9dee-3ac213ab413b" />

Done, now you can open your game and every world has the overlay (you can change the options in the qol settings)
